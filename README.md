# SISTEM MANAJEMEN LAUNDRY

Proyek ini dibangun menggunakan PHP dan framework Bootsrap. Memiliki hak akses multi-level yakni untuk user level admin dan pengelola.

## Fitur-Fitur

1. Form Login untuk Admin dan Pengelola.

2. Menu: Member, Pegawai, Transaksi, Transaksi Operasional, Order, Laporan, Jenis Layanan, Data Satuan, dan Pengaturan.
   
3. Form untuk mengecek status laundry.

## Tools yang Dibutuhkan

1. Teks Editor: **Visual Studio Code**, unduh dan install dari [situs resmi Visual Studio Code](https://code.visualstudio.com/)

2. Web Server: **XAMPP** , dapat diunduh [di sini](https://www.apachefriends.org/download.html)
